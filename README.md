# EstudoLaravel
Projeto de estudo com lista de contatos
## Pré-requisitos
- composer
- npm
- Base de dados postgres (usuário postgres, senha root, banco seller_contacts)
## Rodar frontend
### Entra na pasta frontend e rodar
- $ npm install
- $ npm run dev

## Rodar backend
### Entra na pasta backend e rodar
- $ composer install
- $ php artisan serve


# Dsafio dos múltiplos
Basta rodar node multiplos.js
